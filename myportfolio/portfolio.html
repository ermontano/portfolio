Mimiz

A fun cat-themed maze game created using Python and Pygame. It generates mazes automatically with Prim's algorithm. Mimiz, the character in the game, explores these mazes using depth-first search techniques.

Technologies Used

  • Python backend framework
  • CSS for frontend
  • Pygame backend framework

Key Features

  • 4 Different Character Selection
  • Difficulty Selection
  • End Game Score Identification

SupplAi

App that helps restaurants and fruit suppliers manage their inventory using AI. The app uses AI for image recognition to check the freshness of fruits, predict spoilage, and help businesses keep track of their stock to reduce waste. This app was developed during a hackathon and our team placed 8th out of 18 groups.

Technologies Used

  • Vue.js for frontend
  • JavaScript
  • CSS for frontend
  • HTML
  • Postgress for data storage

Key Features

  • Stock Management
  • Employee Management
  • AI Integrated Image Recognition
